# COBOL Examples 
This is a collection of example and test COBOL programs I've written. I'm currently in the process of updating 
each folder with a README.md file and more comments so that the examples are easier to follow along with.

## Getting Started
To compile and run any example program:
1. Navigate to the specific example directory
2. Compile using: `cobc -x program_name.cbl`
3. Run the compiled program: `./program_name`

All programs were written using [GnuCOBOL](https://gnucobol.sourceforge.io/) in Linux.
